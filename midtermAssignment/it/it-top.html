  • Windowsとは

Microsoft Windows（マイクロソフト ウィンドウズ）は、マイクロソフトが開発・販売するオペレーティングシステム (OS) の製品群である。グラフィカルユーザインタフェース (GUI) を採用している。Windows発売以前では 高価なワークステーション（ハイエンドパソコンを上回る性能のデスクトップコンピュータ）でしか 実現されていなかったマルチタスクやGUIを中心とした使い勝手の良さを、 一般消費者が入手しやすい標準的な規格のパソコンに順次取り込んで行き、 一般向けOSのシェアのほとんどを占めるに至り、今や大きな知名度を持つ。 (Microsoft公式サイト)

  • Windowsの歴史

1985年
Windows 1.0		 最初のWindowsは、DOS上で動作する。 第1段とあって性能はあまり良くない。
1987年
Windows 2.0		 こちらもDOS上で動作。 Windows 1.0から大きく変わったのは、画面がそれまでのタイル型からオーバーラップ型になったこと。 これにより、今では当たり前の「複数のウィンドウ（画面）を重ねて表示すること」ができるようになった。 また、機能改良によってWindows 1.0の課題であった動作の重さが解消され、処理が高速化した。
1990年
Windows 3.0		 このバージョンも以前と同じくDOS上で動作する。 Windows 3.0では次のような機能追加により、以前よりも実用的なOS としてアメリカで一気に普及した。
・複数のMS-DOSアプリケーションをマルチタスク化
・プログラムマネージャー、ファイルマネージャーの導入
・ネットワーク機能
1992年
Windows 3.1		 こちらもDOS上で動作する。 このバージョンではマルチメディア（動画や音声）機能が搭載。さらに安定性が向上し、 より実用性が高まった。これらの改善は、Windows OSのビジネス市場からエンターテイメント市場への 拡大のきっかけになったとも言われている。
1994~96年
Windows NTシリーズ		 「Windows NT」シリーズは、現代のWindowsOSの直接の祖先にあたる。 1994年に「Windows NT 3.1」、1995年に「Windows NT 3.5」、1996年に「Windows NT 4.0」が発売された。 NTシリーズは当初、企業向けの製品に搭載されていたものだが、 NT系の高い安定性が評価されたことで一般向けの製品にも採用されるようになった。
1995年
Windows 95		 Windows 95は、Windows 3.1の性能を引き継ぎ、改良を加えて開発されたWindows 9x系OS。 Startボタン、メニュー、タスクバー、ウィンドウの最大・最小化などの今でも日常的に使っている様々な 便利機能が搭載されたのも、このWindows 95からである。 これまでのバージョンから、機能性・実用性を向上させたWindows 95は、 発売4日で400万本を売り上げを記録。これ以降に開発されるOSの標準としての地位を獲得した。
1998年
Windows 98		 Windows 95をさらに個人消費者向けに改良したのが、1998年に発売されたWindows 98。 全体的なパフォーマンス向上のほか、DirectX5.0搭載、USB・DVD・マルチモニターへの対応などが特徴。 安定したOSとして高い売上を記録している。
2000年
Windows 2000		 Windows 2000は、安定性・堅牢性に優れたWindows NT 4.0を基に構築されたOS。 開発当初、Windows NT系とWindows 9x系の統合が計画されていたことから、 Windows 9x系のユーザーインターフェイスも採用されている
2000年
Windows ME		 Windows 2000が業務用OSとして位置づけされたことにより、急遽リリースされたOS。 短い開発期間の中で多機能性を追求した結果、バグが多くクラッシュも頻繁に発生していた。
2001年
Windows XP		 不評に終わったMEから一変、多くのユーザーから長く愛されるロングセラーとなったOS。 XPはexperience（経験、体験）に由来している。 NT系の安定性と堅牢性、9x系の多機能性と操作性が1つに統合されたWindows XP。 見た目のデザインを一新するなど、様々な改良が加えられた。
2007年
Windows Vista		 多くの新機能が搭載され、画面のデザインも大幅に変更された。 これまでのWindowsとは違った印象をユーザーに与えることができた一方、 操作手順も変わってしまったことで「使いづらい」という評価を受けた。 また、セキュリティ機能の強化やGUIの向上など、スペックを急激に上げてしまったことで、 動作が重くなってしまったことも悪評の要因とされている。
2009年
Windows 7		 2009年に発売されたWindows 7。長い歳月をかけた結果、 不発に終わってしまったWindows Vistaを改良し「これが本来あるべき姿だ」と評価されているOS。
2012年
Windows 8		 スマートフォンやタブレットが普及する中に開発・発売されたOS。 タブレット端末を意識し、タッチスクリーンデバイスへの対応に力を入れたが、 スタートボタンの廃止など斬新すぎるデザインに「前の方が良かった」という声も多数上がった。
2015年
Windows 10		 ”Windows as a Service (WaaS)”という新しいコンセプトの元に開発されたOS。 ローリングリリースモデルが採用され、今後は機能改善や新機能追加などのアップデータはWindows Updateを 通じて無償で行われることに。Windows 10が継続して更新されていくことになるため、 従来のようなメジャーアップデートはなくなった。これについて、マイクロソフトのデベロッパー エバンジェリストのジェリー・ニクソンは「Windows 10はWindowsの最後のバージョン」と述べている。 機能面については、Windows 8で一度削除されたスタート メニューが復活。 音声認識アシスタント機能「Cortana」などの先進的な機能が搭載された。
2021年
